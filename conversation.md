### 📂 What was created

1. **[`render_coa.py`](file:///home/ms/.gemini/antigravity/scratch/furca_coa/render_coa.py)**: The main Python renderer. It supports:
   - **Auto-Scaling:** Automatically detects your terminal dimensions and scales to fit.
   - **Half-Block Rendering:** Uses Unicode `▀` and `▄` characters to double the vertical resolution and preserve a 1:1 pixel aspect ratio.
   - **ANSI State Caching:** Optimizes rendering strings by only outputting escape codes when color changes, which reduces payload size by ~90% and prevents line wrapping.
   - **Color Modes:** Supports 24-bit TrueColor, 256-color, 16-color, and pure high-contrast ASCII art.
   - **History Mode:** Displays the historical background of the name "Furča" and the symbolism behind the wedges.
2. **[`coa.svg`](file:///home/ms/.gemini/antigravity/scratch/furca_coa/coa.svg)**: The original, high-quality vector source SVG file.
